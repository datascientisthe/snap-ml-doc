class snap_ml.RidgeRegression.RidgeRegression(max_iter=50, regularizer=1.0, device_id=0, verbose=False, use_gpu=True, class_weight=None, dual=True)¶

Ridge Regression classifier

This class implements regularized ridge regression using the IBM Snap ML solver. It currently handles sparse matrix input. Use CSR or CSC matrix format for training and CSR format for prediction.

Parameters:
  • max_iter (int, default : 50) – Maximum number of iterations used by the solver to converge.
  • regularizer (float, default : 1.0) – Regularization strength. It must be a positive float. Larger regularization values imply stronger regularization.
  • use_gpu (bool, default : True) – Flag for indicating the hardware platform used for training. If True, the training is performed using the GPU. If False, the training is performed using the CPU.
  • device_id (int, default : 0) – If use_gpu is True, it indicates the ID of the GPU used for training.
  • class_weight ('balanced' or None, optional) – If set to ‘None’, all classes will have weight 1.
  • dual (bool, default : True) – Dual or primal formulation. Recommendation: if n_samples > n_features use dual=True.
coef_¶

array-like, shape (n_features,) – Coefficients of the features in the trained model.

  • y_train (array-like, shape = (n_samples,)) – The target vector corresponding to X_train. The labels should be {-1, 1} or {0, 1}. If the labels are {0, 1} they will automatically be converted to {-1, 1}.
